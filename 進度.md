# QuickBroadcastor 當前進度

## 已完成
- CMake 項目可建置，當前目標為 `QuickBroadcastor`，Debug 產物位於 `BUILD/QuickBroadcastor.exe`。
- GUI 使用 FLTK 1.4，`SendWindow` 已改用 `Fl_Flex` 佈局，不再手寫控件絕對坐標。
- `NotifyWindow` 已實作無邊框右下角通知，顯示發送者與內容，8 秒後自動隱藏。
- `SendWindow` 包含暱稱、消息、多行輸入、發送、隱藏、測試通知按鈕。
- 「測試通知」按鈕會用當前暱稱觸發一條固定測試消息，用於驗證 `NotifyWindow`。
- 托盤管理已改用 `zserge/tray`，由 `TrayManager` 包裝，不再直接在托盤代碼中調用 WinAPI。
- `main` 中 `NotifyWindow`、`SendWindow`、`TrayManager` 均使用棧對象，不使用動態內存。
- 已接入 `vendor/Eati/Essentials`，並通過 vcpkg 增加 `ms-gsl` 依賴。
- 代碼已按最新策劃案風格初步調整：PascalCase 類名、camelCase 函數/變量、指針/引用符號靠右、回調指針使用 `UNWRAP`。

## 近期改動文件
- `src/main.cxx`：創建窗口和托盤管理器，連接發送/測試通知回調。

- `src/trayMan.hxx` / `src/trayMan.cxx`：跨平台托盤包裝。
- `src/stdafx.h`：FLTK、EatiEssentials、GSL 相關公共 include。
- `CMakeLists.txt`：FLTK、zserge/tray、EatiEssentials 接入。
- `vcpkg.json`：`ms-gsl` 依賴。
